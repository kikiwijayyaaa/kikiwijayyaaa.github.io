Services

UI / UX Design

UI (User Interface) dan UX (User Experience) adalah dua konsep penting dalam desain dan pengembang produk digital, seperti aplikasi, situs web, dan perangkat lunak.

Frontend Development

Frontend Development adalah proses pembuatan dan pengembangan bagian depan (front-end) dari sebuah aplikasi web atau mobile, yang berfokus pada pengalaman pengguna (User Experience) dan antar muka pengguna (User Interface).

Backend Development

Backend Development adalah proses pembuatan dan pengembangan bagian belakang (beckend) dari sebuah aplikasi web atau mobile, yang berfokus pada pengelolaan data, logika bisnis, dan integrasi dengan sistem lain.
